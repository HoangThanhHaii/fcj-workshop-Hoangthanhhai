# Harvest Report "EVENT 06/06"

### Purpose of the Event
- Sharing knowledge from seniors, the event was organized to provide the developer community and IT students with a comprehensive overview of the process of developing, deploying, and securing modern applications on cloud platforms.

### List of Speakers

- **Bao Huynh** - Docker - A containerization technology
- **Le Hoang Gia Dai** - Combining AWS WAF with Machine Learning for Cyber Attack Detection on AWS
- **Nguyen Quoc Bao** - Topic: Multiplayer in the Cloud: Connecting Godot Clients with AWS WebSockets
- **Truong Phuoc** - Topic: Effective teamwork methods
- **Viet Phat** - Topic: AWS Neptune for Building a Graph Knowledge Base for GraphRAG
- **Vinh Tran** - Topic: From IT Helpdesk to Senior Sysadmin: Self-study journey and Career transition roadmap to Cloud/DevOps

### Key Highlights
- Infrastructure Transformation & Application Containerization:
Mastering the core concept of Container packaging technology with Docker—an indispensable foundation that standardizes development environments, allowing systems to easily scale and migrate to the Cloud.
- Intelligent Cyber Security with AI/ML:
Exploring next-generation security methods: Combining AWS WAF with Machine Learning models to automate the identification, analysis, and prevention of complex cyber attacks in real time.
- Multiplayer Game Systems on Cloud:
Decoding the low-latency real-time connection architecture between Godot game clients and the AWS cloud via WebSockets, solving the scalability problem for Multiplayer games.
- Heading Advanced AI Trends:
Updating Graph Knowledge Base technology on AWS Neptune to optimize the GraphRAG model—a solution that enables AI/LLM applications to retrieve data intelligently, accurately, and with better context.
- IT Career Advancement Roadmap:
Listening to practical experiences regarding self-study journeys and critical milestones to successfully transition from an IT Helpdesk position to Senior Sysadmin and move towards Cloud/DevOps Engineer.
- Soft Skills for Tech Engineers:
Secrets to effective collaboration among roles (Dev, Ops, Security, Data) within a tech team, helping optimize workflows and accelerate product release speeds.

### Key Learnings
- Knowledge gained after the sharing sessions with the speakers:
+ Mindset, skills, and techniques: Standardizing the Docker development process, knowing how to package applications into containers to ensure software runs consistently across all environments (from local machines and test environments to Production on the Cloud).
+ Career Development Roadmap: Strategic promotion from Helpdesk -> Senior Sysadmin -> Cloud/DevOps, orienting effective methodologies in the IT industry, identifying core skills and required certifications to prepare.
+ Learning how to communicate and intersect work between different departments.
+ Smooth coordination process that helps minimize conflicts and optimize products.

### Work Application
- Application Containerization (Docker): Applying Docker to package microservices or web/game applications. No more "it works on my machine but breaks on the server" scenarios. Easily share the Dev environment with new colleagues with just a single `docker-compose up` command.
- Developing Real-time Features: Applying the WebSockets connection model via the Cloud to build real-time interaction features: multiplayer games, chat systems, notifications, or real-time data updating dashboards.

### Event Experience
- Participating in the event was very beneficial, helping me gain more knowledge, a comprehensive view of future work, and an overview of the future workplace environment.

#### Lessons Learned
- The event serves as a comprehensive bridge helping tech engineers break through their capabilities: from mastering modern infrastructure (Docker, AWS WebSockets), optimizing security, and applying next-gen AI (AWS WAF + ML, GraphRAG with AWS Neptune), to practicing teamwork skills and shaping a practical career advancement roadmap from IT Helpdesk to Cloud/DevOps Engineer.
